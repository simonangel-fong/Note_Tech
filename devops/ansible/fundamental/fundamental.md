## Ansible

### Ansbile parameters

- The parameter used to define how Ansible runs playbooks
- Naming:
  - `ANSIBLE_<PARAMETER_NAME>`: All upper case.

---

## Idempotency

- `Idempotency`
  - ensures that **executing a task multiple times** yields the **exact same system state** without producing unintended side effects.
