## Jinja2

- `Jinja2`:
  - a Python-based templating engine that Ansible uses to **dynamically generate configuration files**, manipulate text, and insert variable values during playbook execution.
  - template file extension `.j2`

- Syntax:

- `{{ }}`: print out variable values or the results of a calculation.
- `{% %}`: Used for programming logic, such as loops, conditionals, and control flow.
- `{# #}`: Comments
