## Common Services

- Computing:
  - execuing code in the cloud

- Common Services:
  - `Virtual Machine(VM)`:
    - a normal cloud server.
    - ~= `AWS EC2`
  - `VM Scale Sets(VMSS)`:
    - a group of similar VMs managed together.
    - ≈ `EC2 Auto Scaling Group`
  - `App services(Web apps)`:
    - a managed platform for hosting web applications and APIs.
    - ≈ AWS Elastic Beanstalk / App Runner / managed web hosting
  - `Azure Container Instances(ACI)`
    - run a container in Azure.
    - ≈ `ECS RunTask` with Fargate-like simplicity
  - `Azure Container Apps`
    - a serverless container application platform.
    - AWS App Runner / ECS Fargate-like platform
  - `Azure Kubernetes Service(AKS)`
    - managed Kubernetes on Azure.
    - ≈ EKS
  - `Azure Virtual Desktop`
    - providing remote desktops and remote apps to users.
    - ≈ Amazon WorkSpaces / AppStream
