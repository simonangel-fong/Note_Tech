## Structure 结构

- /root: 根目录
  - README.md: 记录介绍信息
  - index.md: 笔记page的首页
  - /rep/: 目录，用于存储所有笔记和实例
  - /tech: 目录，用于存储具体的技术
    - tech_index.md：技术首页
    - /app/: 目录，用于存储技术的子页面
    - /pics/: 目录，用于存储截图
    - /lab/: 目录，用于存储具体的实例

```mermaid
graph TD

A[root]
    A --> B(README.md)
    A --> C(index.md)
    A --> E[tech]
    A --> D[rep]
        E --> F(tech_index.md)
        E --> G[app]
        E --> H[pics]
        E --> I[lab]
```
