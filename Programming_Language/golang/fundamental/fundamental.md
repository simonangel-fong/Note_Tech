## Syntax

- Go file consists of the following parts:
  - Package declaration: `package <pkg_name>`
  - Import packages: `import ("fmt")`
  - Functions: `func main() {}`
  - Statements and expressions

- Go statements are separated by
  - ending a line
  - a semicolon ";"

- Go comments
  - Single-line comments: `// This is a comment`
  - Multi-line comments: `/* This is a comment */`

---

- `Static Type language`
  - checks variable types **before** the program runs.
  - variable type is known **at compile time**.
  - c++, java, go, Rust, TypeScript

- `Dynamic Type language`
  - checks variable types **while** the program is running.
  - the variable type is checked **when the program runs**.
  - js, ruby, python, PHP
