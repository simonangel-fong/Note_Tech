## Setup and Configuration

| Command                                        | Description                  |
| ---------------------------------------------- | ---------------------------- |
| `git config --list`                            | Show all Git config settings |
| `git config --global user.name "owner_name"`   | Set owner for commits        |
| `git config --global user.email "owner_email"` | Set email for commits        |
